## spinner

```
USAGE: spinner.sh [OPTIONS] <command>

Run a command with an animated spinner.

OPTIONS:
  -d            Enable debug mode.
  -s <spinner>  Specify a spinner to use. Default is "line".
                See ./spinners.json for a list of available spinners.
  -h            Print this message and exit.
```
